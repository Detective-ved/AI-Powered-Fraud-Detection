# 🛡️ Smart AI-Powered Fraud Detection System

A Machine Learning project that intelligently detects fraudulent transactions using multiple classification algorithms and data balancing techniques.

---

## ✨ Highlights

* 📊 Data Cleaning & Preprocessing
* 🧹 Missing Value Handling
* 🏷️ Label Encoding
* ⚖️ Feature Scaling
* 🔄 SMOTE for Class Balancing
* 🤖 Logistic Regression
* 🌲 Random Forest Classifier
* 🚀 XGBoost Classifier
* 📈 Model Evaluation
* 📉 Confusion Matrix Visualization
* 🎯 ROC-AUC Score Calculation

---

## 🛠 Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost
* Imbalanced-Learn (SMOTE)

---

## 📌 Workflow

Dataset → Data Cleaning → Feature Engineering → SMOTE → Train/Test Split → Model Training → Fraud Prediction → Performance Evaluation

## 📊 Evaluation Metrics
